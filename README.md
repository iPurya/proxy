# Usage :

## Set permission
`chmod +x purya`

## run
`./purya <SECRET> <TAG>`

For <TAG>, you must put you admin tag from @MTProxybot.
and for <SECRET> you can create a random hash with:
`head -c 16 /dev/urandom | xxd -ps`
 
 
 ## Example
`./purya d1499a84da579c09ab966656c5ff9b09 097470b6581ef963eea3e4fa385bbe5c`
